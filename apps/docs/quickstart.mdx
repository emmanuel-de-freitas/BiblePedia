## Installation

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/emmanuel-de-freitas/biblepedia.git
    cd biblepedia
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    bun install
    ```
  </Step>

  <Step title="Set Up Environment">
    Decrypt the environment variables (requires decryption key):
    ```bash
    bun run env:decrypt
    ```
    
    If you don't have the key, copy from the example:
    ```bash
    cp .env.example .env
    ```
  </Step>

  <Step title="Start Development">
    Choose your development mode:
    
    <Tabs>
      <Tab title="Desktop App">
        Start both web and desktop together:
        ```bash
        bun run desktop
        ```
      </Tab>
      <Tab title="Web Only">
        Start just the web app:
        ```bash
        cd apps/web
        bun run dev
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>

<Check>
  **Success!** You should see the desktop app window open or the web app running at `http://localhost:5173`
</Check>

## Troubleshooting

<AccordionGroup>
  <Accordion title="bun: command not found">
    Install Bun using the official installer:
    ```bash
    curl -fsSL https://bun.sh/install | bash
    ```
  </Accordion>

  <Accordion title="Environment variables not loading">
    Ensure you've decrypted the `.env` file or copied from `.env.example`:
    ```bash
    bun run env:decrypt
    # OR
    cp .env.example .env
    ```
  </Accordion>

  <Accordion title="Desktop windows not appearing">
    Make sure the Vite dev server is running first:
    ```bash
    cd apps/web && bun run dev
    ```
    Then in another terminal:
    ```bash
    cd apps/desktop && bun run dev
    ```
  </Accordion>
</AccordionGroup>
