## Quick Start

Get up and running in minutes with our comprehensive guides.

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install dependencies using Bun:
    ```bash
    git clone https://github.com/emmanuel-de-freitas/biblepedia.git
    cd biblepedia
    bun install
    ```
  </Step>
  <Step title="Set Up Environment">
    Decrypt environment variables for local development:
    ```bash
    bun run env:decrypt
    ```
  </Step>
  <Step title="Start Development">
    Launch the desktop app with hot reload:
    ```bash
    bun run desktop
    ```
  </Step>
</Steps>

## Technology Stack

BiblePedia is built with modern, cutting-edge technologies:

<Accordion title="Core Runtime & Build">
  - **Bun** - JavaScript runtime and package manager (replaces Node.js)
  - **TypeScript 6** - Type-safe development
  - **Turborepo** - Monorepo build system with smart caching
  - **Biome** - Fast code formatting and linting
</Accordion>

<Accordion title="Desktop & Web">
  - **Electrobun** - Desktop framework with native webviews
  - **React Router 7** - Full-stack React framework with SSR
  - **Vite** - Lightning-fast development server
</Accordion>

<Accordion title="UI & Styling">
  - **React Spectrum S2** - Adobe's design system
  - **HeroUI** - React component library
  - **Tailwind CSS** - Utility-first CSS framework
  - **Motion** - Animation library
</Accordion>

<Accordion title="State & Data">
  - **Jotai** - Atomic state management
  - **dotenvx** - Encrypted environment variables
</Accordion>
